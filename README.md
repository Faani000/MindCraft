# MindCraft

MindCraft is a collection of Claude Code skills designed to craft clarity, structure, and efficiency from everyday cognitive tasks.

## Purpose
This repository demonstrates how everyday human work can be extracted into small, reusable Claude Code skills that reduce mental load and improve productivity.

## Included Skills
- **Concept Simplifier** – Explains complex topics in clear, beginner-friendly language.
- **Notes Condenser** - Converts long notes or text into short , clear , well-structured summaries.
- **MCQ Generator** – Creates high-quality multiple-choice questions for learning and assessment.

## How These Skills Help
Each skill:
- Replaces a repeatable cognitive task
- Produces one clear, measurable outcome
- Can be reused by others with similar needs
