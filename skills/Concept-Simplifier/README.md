# Concept Simplifier Skill

This skill helps students break down complex academic concepts into simpler, more understandable parts.

## What it does

The concept-simplifier skill takes a complex topic and returns:
1. A simple, beginner-friendly explanation (under 150 words)
2. A real-life example to illustrate the concept
3. Three key points summary

## How to use

Provide the name of the concept or topic you want simplified, and the skill will return a clear, structured explanation following these constraints:
- Uses simple English
- Explanation under 150 words
- Output in bullet points or numbered format
