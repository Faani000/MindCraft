# Notes Condenser Skill

This skill helps students condense long study materials into short, focused notes that are easy to review and remember.

## Overview

The notes-condenser skill transforms lengthy study materials into concise, focused notes by extracting the most important information and presenting it in a clear, bullet-point format for efficient studying.

## Features

- Condenses long text into summaries under 150 words
- Uses simple, clear English
- Outputs information in bullet points only
- Highlights key points for easy reference
- Preserves essential information while removing unnecessary details

## How to Use

1. Prepare your long text or study material
2. Provide the text to the skill
3. Receive a condensed version with key points highlighted

## Input

- Long text or study material (string) - The content that needs to be condensed into shorter notes

## Output

1. Short bullet point summary of the content (under 150 words)
2. Key points highlighted in bullet points

## Constraints

- Output is limited to 150 words maximum
- Uses simple, clear English
- All output is provided in bullet points format only
- Preserves essential information while condensing

## Benefits

- Saves study time by focusing on essential information
- Makes review sessions more efficient
- Helps identify key concepts quickly
- Creates portable, easy-to-read notes
- Maintains the core meaning of the original content
