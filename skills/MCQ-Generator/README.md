# MCQ Generator Skill

This skill helps students generate multiple choice questions for practice based on a specified topic, enabling effective self-assessment and review.

## Overview

The mcq-generator skill creates beginner-friendly multiple choice questions with clear options and marked answers. This allows students to test their understanding of concepts and practice for exams.

## Features

- Generates beginner-friendly MCQs based on any topic
- Creates four options (A, B, C, D) for each question
- Clearly marks the correct answer for each question
- Uses simple, clear language
- Avoids ambiguous or trick questions
- Allows specifying the number of questions to generate

## How to Use

1. Choose a topic or concept you want to practice
2. Specify how many questions you want to generate
3. Receive MCQs with four options and marked answers

## Input

1. Topic name or short concept (string) - The subject area for the MCQs
2. Number of MCQs (integer) - How many questions to generate (1-20)

## Output

1. Multiple choice questions (MCQs)
2. Four options (A, B, C, D) for each question
3. Correct Answer clearly marked for each question

## Constraints

- Questions are beginner-friendly and not ambiguous
- No trick questions are included
- Correct answers are clearly marked
- Language is kept simple and clear
- Each question has exactly four options (A, B, C, D)

## Benefits

- Creates personalized practice tests
- Helps identify knowledge gaps
- Enables self-assessment of understanding
- Provides immediate feedback with correct answers
- Uses appropriate difficulty level for beginners
